# Iodine UI (WIP) 🎚️
Iodine UI is a Material Design inspired UI library for Vue 3 with a focus on interactive components

## Project setup (standard) ⚙️
```bash
# Install
npm install
# Serve
npm run serve
# Or build
npm run build
```
